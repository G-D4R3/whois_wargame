# wargame

## 개요
아주대학교 사이버보안학과 정보보안소학회 Whois 내부 wargame site front-end

### stack
Vue.js

## 기능
- 회원가입, 로그인
- 스코어보드
- wargame 문제 표시 및 flag 제출 확인
